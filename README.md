# create-nextjs

# Install bun

```bash
curl -fsSL https://bun.sh/install | bash
```

# Create template

```bash
bun create kmartinezmedia/create-nextjs
bun install
```

# Dev

```bash
bun run dev
```

# Build

```shell
bun run build
```

# Lint

```shell
bun run lint
```

# Format

```shell
bun run format
```
